# Digital Watch

A simple digital watch that displays current time and date.

## Features
- Shows time in 12-hour format (with AM/PM)
- Shows full date (day, month, day number, year)
- Updates every second
- Dark theme design

## How to Use
1. Open `index.html` in any web browser
2. The time and date will update automatically

## Technologies Used
- HTML5
- CSS3
- JavaScript
